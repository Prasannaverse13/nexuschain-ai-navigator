# NexusChain AI Navigator

This is a Next.js project demonstrating a multi-agent AI system for supply chain analysis, built in Firebase Studio.

## How to Run

To get started, run the development server:

```bash
npm run dev
```

This will start the Next.js application on port 9002.

To run the AI agent backend (Genkit), use a separate terminal:

```bash
npm run genkit:watch
```
